Marca Exclusiva

Scripts e Procedimentos de apoio para o programa da Marca Exclusiva.

⭐ Regras e Níveis do Programa (2026)

Tabela comparativa atualizada para impressão sob demanda.

Nível Volume (Pares/Mês) Tipo de Certificado / Custo Prazo de Produção
Essencial 8 a 49 pares Padrão (várias cores) ou Arte Própria (Custo do Cliente) 4 dias úteis
Superior 50 a 99 pares Padrão (várias cores) ou Arte Própria (Pago pela Haytek) 4 dias úteis
Elite 100+ pares Padrão, Arte Própria ou Personalizada (Pago pela Haytek) 4 dias úteis (20 dias se personalizada)

⚠️ Pré-Marca Exclusiva (Teste de Volume)

Todo novo parceiro que iniciar nos níveis Superior ou Elite passará por um período de teste de 3 meses (90 dias).

💲 Regras de Cobrança e Tabelas (Custo R$ 8,00)

  • Nível Essencial: Se optar por Arte Própria, terá o custo fixo de R$ 8,00.
    • Foram criadas tabelas de preços específicas para serem aplicadas nestes clientes: Padrão, 5% e 10%.
  • Níveis Superior e Elite: Caso não atinjam as quantidades necessárias após o período de teste de 90 dias, passarão a ter a cobrança de R$ 8,00. (O acompanhamento e sinalização para o Cadastro incluir esses valores partirá do Marketing).
  • Tabelas Negociadas (Atenção): Caso o cliente tenha tabela negociada em algum produto, o valor de R$ 8,00 terá que ser calculado e embutido para inclusão dos valores.
  • Tabelas Base: Sempre que o cliente tiver tabela negociada, consulte se se trata de alguma tabela Base. Se sim, retire a tabela negociada e inclua na base a tabela correta com o acréscimo devido.
  • Exceção: Se for preciso criar uma nova tabela base, por favor, acione a liderança.
📋 Roteiro do Processo da Marca Exclusiva

🧑‍💼 Vendedor

  • Consulta a planilha: Consulta Marcação Sigla Marca Exclusiva.
  • Coloca os dados do cliente no formulário: Acessar Formulário.

💻 Cadastro

  • Copia os dados da planilha Cadastro no Programa de Marca Exclusiva (Respostas) para a planilha Registro Marca Exclusiva.
  • Os dados cadastrais do cliente retire do CRM.
  • Confirme se o código é master.
  • Se o código for rede, confirme se deve bloquear as siglas para todos os estados.
    • Se for bloquear para todos os estados:
      Na planilha de Registro, aba "KA", coloque o código.
      Na planilha Consulta Marcação Sigla Marca Exclusiva, aba "Bloqueadas(oculta)", coloque as siglas.
  • Envie o contrato para o cliente e vendedor via Docusign.
  • Enviar e-mail com Script constante em: Contrato Enviado.
    • Título: Marca Exclusiva - [Código do cliente]
    • Destinatários: Quem enviou a solicitação; cadastro@haytek.com.br; marcaexclusiva@haytek.com.br
  • Confira junto ao Docusign diariamente a assinatura do contrato.
  • Após todos os envolvidos assinarem o contrato encaminhar para o laboratório: marcapropria@haytek.com.br.
    • Script: Contrato Assinado - Marca Exclusiva (Enviar ao Lab)
    • Dica: Usar mesmo e-mail que informou o envio do contrato, mantendo todos que já constam em cópia e inclua o e-mail: marcapropria@haytek.com.br.
  • Na planilha de Registro coloque na coluna "Data da Reserva" a data que o contrato foi assinado.
  • Ao enviar para o Lab, configure no CRM com as informações da Marca no código master e pode replicar para os filhos.

🌟 Quando Nível Superior ou Elite

  • (Arte própria ou personalizada):
    Após o Laboratório dar ok na configuração da marca, encaminhar para o Marketing: marketing.ticket@haytek.com.br.
📖 Regras e Procedimentos Adicionais
Siglas Indisponíveis para Uso:
Enviar e-mail para o vendedor anexando a planilha de consulta das siglas para verificação das siglas disponíveis e reenviar (no próprio e-mail).
Níveis que Precisam de Aprovação por E-mail:
Superior e Elite (Aprovação via diretoria comercial ou gerente nacional).
Entrada de Rede na Marca:
O processo inicial é o mesmo de todos. Caso seja uma rede com mais de 30 códigos:
  • Envie para o Lab para o cadastro do código principal.
  • Os demais códigos: gere relatórios com todos os códigos e abra ticket para que a equipe da Monique inclua de forma massiva internamente.
Alteração ou Inclusão:
A alteração ou inclusão só exige reenvio de contrato caso o cliente esteja migrando para os novos perfis de níveis da marca.
Caso contrário: Após o vendedor colocar no formulário da marca, incluímos as alterações na planilha de registro e encaminhamos para o Lab.
Quando Enviar para o Lab:
Apenas em alterações das siglas, nome da marca, certificado ou código master cadastrado na marca. (Alteração apenas de nível não é necessário incluir o laboratório).
Certificado Provisório:
Faz-se necessário caso o cliente ainda não tenha a marca registrada. Caso contrário, mantém o cartão já utilizado.
Tratativa em Erro no Envio dos Certificados:
  • 1º Solicite número de um dos pedidos com erro.
  • 2º Consulte a data que o cliente entrou na marca e a data do pedido.
  • 3º Consulte se o cliente tem marca no produto do certificado.
  • 4º Lentes Transitions não têm marca exclusiva.
  • 5º Consulte a marca cadastrada pelo laboratório no Painel Logística.

Se todas as etapas acima estão de acordo: Envie para o laboratório para análise e reenvio do certificado correto.

Atenção: Se o pedido foi colocado antes da configuração da marca em sistema, o cliente receberá certificado Haytek (mesmo o faturamento sendo posterior à inclusão do pedido).

Regra de Ouro: Não enviamos certificados Haytek para clientes com marca exclusiva. Casos excepcionais devem acionar a liderança.
